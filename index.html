Packaging Database (Procter and Gamble)

Implemented a global packaging database, to enable eco savings and strategic packaging planning. Access database with Visual Basic GUI, data also automatically pulled from SAP stock control database using SQL. Queries written in SQL. Drawings Browser (Scott Wilson Railways) - Industrial Placement Project

Implemented a GUI for searching and viewing engineering drawings, implemented in Visual Basic with a backend database built using Access. Queries built on the fly using SQL based on user input which then returns the correct PDF/PNG drawing on the system for the engineer to inspect. 60000+ drawings indexed on the system, retrieval from the network within seconds based on metadata stored with the scanned prints. Replaced paper system completely, fully future proofed company drawings system.
